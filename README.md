# Консольная игра Слоты 3x3

Игра-автомат с сеткой 3x3 символов для консоли.

## Запуск игры

```bash
npm run game
```

## Правила игры

- **Горизонтальная линия**: выигрыш x3 от ставки
- **Вертикальная линия**: выигрыш x3 от ставки  
- **Диагональ**: выигрыш x5 от ставки
- **Все символы одинаковые**: бонус x10 от ставки
- Можно получить несколько выигрышей одновременно

## Управление

1. Крутить барабаны (SPIN) - запускает игру
2. Изменить ставку - позволяет изменить размер ставки
3. Правила игры - показывает правила
4. Выход - завершает игру

## Начальные условия

- Стартовый баланс: 1000
- Минимальная ставка: 10
